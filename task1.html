Responsive Landing Page

Create an interactive navigation menu that changes style when scrolled or hovered. This responsive landing page is built using HTML, CSS, and JavaScript.

About Task-01

This project demonstrates how to build a responsive landing page with a fixed navigation bar. The navbar changes its background color when the page is scrolled and menu items respond with hover effects.

Technologies Used

HTML is used for the page structure, CSS is used for styling and responsiveness, and JavaScript is used to add interactive effects such as scroll-based navbar changes.
